1. Overview

The WanderWays travel platform uses AWS services to provide an elastic, scalable, and secure environment. It handles everything from user authentication to trip recommendations and bookings through an automated, monitored deployment pipeline.

3. Architecture Components

  • Frontend: S3, CloudFront, Route 53, ACM
  • Backend: EC2 Auto Scaling, Load Balancer, Beanstalk
  • Database: DocumentDB / MongoDB Atlas
  • Auth: Cognito + JWT
  • AI: Lambda + API Gateway (Gemini API)
  • Monitoring: CloudWatch, SNS
  • Security: IAM, WAF, Secrets Manager

4. Network Design

  • VPC: 10.0.0.0/16 (2 public, 2 private subnets)
  • Public: Load Balancer, NAT Gateway
  • Private: EC2, DocumentDB
  • Security Groups + NACLs for traffic control
  • Multi-AZ for high availability

5. Scaling & Monitoring

  • Min instances: 1 | Max: 4
  • Scale out: CPU > 60%
  • Scale in: CPU < 30%
  • Monitored by CloudWatch dashboards

6. Cost Planning (Detailed AWS Monthly Estimate)

The following table provides an estimated monthly cost breakdown for the WanderWay cloud deployment under moderate traffic (~20,000 requests/day, ~10 GB storage). AWS Free Tier benefits are applied where possible.

Layer Service Usage Details Estimated Monthly Cost (USD)
Presentation Layer Amazon S3 Static React site hosting (~5 GB storage, 10 GB data transfer) $1.50
Amazon CloudFront (CDN) 100 GB data transfer (global edge delivery) $8.00
Amazon Route 53 + Certificate Manager 1 hosted zone + 1 SSL certificate $1.00
Application Layer Elastic Beanstalk + EC2 t3.small (2 instances, auto-scaling enabled, 200 hrs/mo) $20.00
Application Load Balancer Handles API traffic and scaling $5.00
AWS Lambda + API Gateway ~1M requests/month (AI recommendation service) $2.00
Data Layer Amazon DocumentDB db.t3.medium instance + 20 GB storage + backup $45.00
Amazon ElastiCache (Redis) Cache layer for frequent queries (cache.t3.micro) $8.00
Amazon Cognito ~1000 active users (authentication & tokens) $5.00
Monitoring & Security Amazon CloudWatch + X-Ray 10 custom metrics + 5GB logs $3.00
AWS WAF Web security (5 rules) $4.00
AWS Secrets Manager Secure storage for API keys, credentials $1.00
Amazon SNS ~10K notifications per month $1.00
Management & CI/CD AWS CodePipeline + CodeDeploy Automated deployment pipeline (GitHub → EC2) $2.00
AWS IAM Access control (no cost under Free Tier) $0.00
Total Estimated Monthly Cost $106.50

Note: Costs may vary based on region and traffic volume. AWS Free Tier provides significant savings for the first 12 months:

  • 750 hrs/month of t2.micro or t3.micro instance usage (free EC2 tier)
  • 5 GB S3 storage & 15 GB CloudFront data transfer free
  • 1M Lambda requests/month free

Using Free Tier + Spot Instances can reduce the total to ~$25–$35/month during development or testing phases.

7. Security & IAM

  • IAM Roles for EC2, Lambda, Pipeline
  • Secrets Manager for credentials
  • WAF for protection
  • HTTPS enforced with ACM

8. Backup & Recovery

  • DocumentDB auto snapshots
  • EC2 AMI snapshots
  • S3 versioning + Glacier

9. CI/CD Workflow

  • GitHub → CodePipeline trigger
  • CodeBuild → Build artifact
  • CodeDeploy → EC2 deployment
  • CloudFront invalidation

10. Key Benefits

  • Scalable and elastic AWS infra
  • High availability with multi-AZ
  • Secure with IAM, WAF, HTTPS
  • Automated CI/CD deployment
  • Pay-as-you-go, Free Tier friendly
